About

This loan processing app is designed to evaluate whether a car loan applicant is credit-worthy of an auto-loan. Several machine learning algorithms were tested. The most accurate model was selected. You can find a Jupyter Notebook which shows the data pre-processing, and model selection at the following link: www.mygithub.

The algorithm used was a Gradient Boosting Classifier, which was uniquely identified as the best performing classifier amongst other classifiers. The following performance metrics are listed below:

  • -- Accuracy: 87.9% --
  • -- ROC-AUD: 95.2% --
  • -- Precision (Approved): 85% --
  • -- Precision (Declined): 90% --
  • -- Recall (Approved): 89% --
  • -- Recall (Declined): 87% --
